# ClinicMate Frontend

This is the frontend for the ClinicMate application, built with React and Vite.

## Prerequisites

- Node.js (v16 or higher)

## Installation

1.  Clone the repository:
    ```bash
    git clone <repository-url>
    cd Frontend
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

## Running the Application

-   **Development Server**:
    ```bash
    npm run dev
    ```
    Open [http://localhost:5173](http://localhost:5173) to view it in the browser.

-   **Mock Server** (for students data):
    ```bash
    npm run server
    ```
    This runs `json-server` on port 5000.

-   **Linting**:
    ```bash
    npm run lint
    ```

-   **Build for Production**:
    ```bash
    npm run build
    ```

-   **Preview Production Build**:
    ```bash
    npm run preview
    ```

## specialized Libraries

-   **UI Components**: Radix UI, Lucide React, FontAwesome
-   **State Management**: Redux Toolkit
-   **Styling**: Tailwind CSS
-   **Networking**: Axios
